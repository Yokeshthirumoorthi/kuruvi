# kuruvi
Copyright (c) 2019 Yokesh Thirumoorthi
WIP.

With kuruvi you can extract Exif metadata from images and persist it to a database.

#### Build and Run Instructions

```
git clone https://github.com/Yokeshthirumoorthi/kuruvi
cd kuruvi
npm start
```
